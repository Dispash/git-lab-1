# Программная реализация

## Определение и контроль тепловой мощности исследовательских реакторов на тепловых нейтронах

Данное веб-приложение предоставляет возможность расчета погрешности для методик расчета тепловой мощности.

Всего представлено **4 методики**:

1. *№1417* Тепловая мощность РБТ-6 по значениям теплотехнических параметров теплоносителя;
1. *№597* Методика расчета тепловой мощности РБТ-10;
1. *№1122* Расчет тепловой мощности СМ-3 по параметрам теплоносителя;
1. *№1341* Определение тепловой мощности реактора МИР.М1.
